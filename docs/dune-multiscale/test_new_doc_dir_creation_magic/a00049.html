Assemble right hand side vectors for all local problems on one coarse cell.

Parameters
[in]coarseEntityThe coarse cell.
[out]allLocalRHSA vector with pointers to the discrete functions for the right hand sides.
Note
The vector allLocalRHS is assumed to have the correct size and contain pointers to all local rhs functions. The discrete functions in allLocalRHS will be cleared in this function.
